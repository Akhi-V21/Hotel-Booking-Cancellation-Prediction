# Hotel Booking Cancellation Prediction

## 📌 Problem Statement
Hotel booking cancellations cause revenue loss. This project predicts whether a booking will be canceled or not using Machine Learning.

## 📊 Dataset
Hotel Booking Demand Dataset

## ⚙️ Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## 🤖 Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## 📈 Results
Random Forest gave the best accuracy.

## 💡 Key Insights
- Higher lead time → more cancellations
- No deposit bookings → higher cancellations
- Repeated guests → less cancellations
